Function List

1. Top Up E-Money

Pengguna dapat melakukan top up e-money dengan memasukkan jumlah saldo yang diinginkan.

3. Check E-Money Balance

Program dapat menampilkan saldo e-money yang dimiliki oleh pengguna.

5. View Transaction History

Pengguna dapat melihat seluruh history transaksi yang dilakukannya.

7. View User Information

Program dapat menampilkan informasi tentang pengguna.

9. User Registration

Pengguna dapat melakukan pendaftaran akun.

2. Transfer Balance

Pengguna dapat melakukan Transfer saldo ke pengguna lain dengan memasukkan nomor penerima dan jumlah saldo yang akan ditransfer.

4. Make Payment

Pengguna dapat melakukan pembayaran dengan memasukkan jumlah yang harus dibayarkan dan saldo e-money yang dimiliki akan otomatis terpotong.

6. Filter Transaction History

Pengguna dapat melihat history yang ditentukan (Filter).

8. Update Personal Data

Pengguna dapat merubah data pribadi.

10. User Login and Logout

Pengguna dapat melakukan login dan logout.
